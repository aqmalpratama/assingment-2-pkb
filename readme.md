#How to run

1. Open file main.py
2. Data yang ada dalam iris.csv itu digunakan untuk input datanya
3. Instalisasi library numpy, pandas, dan scikit-learn

`pip install numpy`
`pip install pandas`
`pip install scikit-learn`

4. Apabila ingin menambahkan data ubah dalam file iris.csv
5. Run the code
